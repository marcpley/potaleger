### Application Base de Données Flexible et Ambitieuse

Développe ton application base de données avec seulement du SQL !

- Application de bureau (pas en ligne).
- Interface utilisateur final toute prête, puissante et paramétrable.
- Logique métier tout en SQL (et un peu de python).
- Base de données SQLite avec tables FADA contenant la logique métier et le paramétrage de l'interface utilisateur.
